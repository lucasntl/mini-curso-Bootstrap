Passo 1: A Estrutura Básica

Todo site Bootstrap começa com este HTML. Apenas copie, cole e adicione o link do CSS no <head> e do JavaScript antes de fechar o </body>.

<!DOCTYPE html>
    <html lang="pt-BR">
    <head>
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <title>Meu Site</title>
        <link href="..." rel="stylesheet">
    </head>
    <body>
        <h1>Olá, mundo!</h1>
        <script src="..."></script>
    </body>
    </html>
